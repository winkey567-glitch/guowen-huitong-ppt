# 🎨 Digital Gallery 高端设计风格指南

## 设计理念

**融合四大元素：极简 + 艺术 + 科技 + 权威**

参考：博物馆/美术馆数字化展示风格

---

## 核心设计规范

### 1. 布局 (极简 + 权威)
- Split Screen 分屏布局
- Asymmetric Grid 不对称网格
- 40% 留白区域
- 背景：Off-White (#F9F9F7) 或 Deep Space Grey (#111111)

### 2. 字体 (艺术 + 权威)
- **标题**: Playfair Display / Cinzel (衬线体)
- **正文**: Inter / Helvetica (无衬线体)
- **字号对比**: 标题 5rem+，正文 0.9rem

### 3. 动效 (科技)
- Staggered Reveal 交错揭示
- cubic-bezier 平滑缓动
- 1px 超细线条分割 (透明度 20%)
- 抽象几何体背景装饰

### 4. 配色
- **主色**: 黑白灰单色系
- **点缀色**: Klein Blue (#002FA7) 或 Crimson Red (#DC143C)
- 仅用于按钮、链接、关键数字

