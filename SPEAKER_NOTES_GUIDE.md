# 国文汇通 PPT 旁白功能使用指南

## 🎯 功能概述

为 `gallery-investor-v2.html` 添加了专业的演讲者备注（旁白）功能，支持多种展示模式和交互方式。

---

## ✨ 核心功能

### 1. **底部抽屉模式**（默认）
- 点击右下角 📝 按钮，从底部弹出旁白抽屉
- 占据屏幕下方 35% 高度
- 毛玻璃效果 + 黑金配色
- 不遮挡主要内容

### 2. **侧边栏模式**
- 点击"切换侧边栏"按钮切换到侧边栏模式
- 从右侧滑出 400px 宽度的旁白面板
- 适合宽屏演示

### 3. **演讲者模式**（专业功能）
- 按 `S` 键打开演讲者视图
- 新窗口显示：
  - 当前页面预览
  - 完整旁白文档
  - 演讲计时器
- 双屏演示的最佳选择

---

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `N` | 切换旁白显示/隐藏 |
| `S` | 打开演讲者模式 |
| `ESC` | 关闭旁白或演讲者模式提示 |

---

## 🎨 设计特点

### 视觉风格
- **黑金配色**：与主PPT风格完全一致
- **毛玻璃效果**：backdrop-filter 实现高级质感
- **金色边框**：rgba(255, 215, 0, 0.3) 点缀
- **平滑动画**：cubic-bezier 缓动函数

### 交互体验
- **自动跟随**：旁白内容随页面滚动自动更新
- **状态指示**：按钮高亮显示当前状态
- **快捷键提示**：首次加载显示 5 秒提示
- **响应式设计**：移动端自动适配

---

## 📝 旁白内容结构

每页旁白包含：
- **页码**：Page X of 24
- **标题**：当前页面主题
- **演讲稿**：详细的演讲者备注

示例：
```
Page 1 of 24
封面 - 国文汇通

各位投资人，大家好。今天我将为大家介绍国文汇通——一个响应国家文化数字化战略的创新平台。我们致力于构建数字文创资产合规发行与交易的一站式生态体系。
```

## 🔧 技术实现

### 文件结构
```
gallery-investor-v2.html    # 主PPT文件（已集成旁白功能）
speaker_notes.json          # 旁白数据（24页完整内容）
add_speaker_notes.py        # 旁白功能生成脚本
```

### 核心技术
- **CSS Transform**：流畅的滑入/滑出动画
- **JavaScript 监听**：scroll 事件自动更新旁白
- **Window.postMessage**：演讲者模式窗口通信
- **JSON 数据驱动**：旁白内容独立管理

## 🎬 使用场景

### 场景 1：投资人路演
1. 打开 `gallery-investor-v2.html`
2. 按 `S` 键打开演讲者模式
3. 主屏幕投影给观众（纯净PPT）
4. 笔记本屏幕显示演讲者视图（含旁白+计时器）

### 场景 2：内部培训
1. 打开 PPT
2. 按 `N` 键展开底部抽屉
3. 边讲解边参考旁白内容
4. 按 `N` 键收起，保持画面简洁

### 场景 3：自学复习
1. 点击"切换侧边栏"按钮
2. 侧边栏常驻显示旁白
3. 滚动浏览，旁白自动更新
4. 完整阅读演讲稿内容

## 📱 响应式支持

### 桌面端（>768px）
- 底部抽屉：35vh 高度
- 侧边栏：400px 宽度
- 所有功能完整可用

### 移动端（≤768px）
- 底部抽屉：50vh 高度
- 侧边栏：全屏宽度
- 自动隐藏模式切换按钮

## 🔄 更新旁白内容

### 方法 1：编辑 JSON 文件
```json
{
  "1": {
    "title": "封面 - 国文汇通",
    "notes": "你的演讲稿内容..."
  }
}
```

### 方法 2：重新运行脚本
```bash
python add_speaker_notes.py
```

---

## 🎨 自定义样式

### 修改颜色
在 CSS 中搜索 `#FFD700`（金色）替换为你的品牌色

### 调整尺寸
- 抽屉高度：`.notes-drawer { height: 35vh; }`
- 侧边栏宽度：`.notes-sidebar { width: 400px; }`

### 修改动画
- 动画时长：`transition: transform 0.4s`
- 缓动函数：`cubic-bezier(0.4, 0, 0.2, 1)`
